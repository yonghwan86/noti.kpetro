# AI 업무 알림 서비스 — 담당자(Staff) 매뉴얼

## 1. 로그인

### 최초 로그인 (비밀번호 미설정 시)
1. 로그인 페이지에서 등록된 이메일 주소를 입력합니다.
2. **"계속"** 버튼을 클릭합니다.
3. 최초 로그인 시 비밀번호 설정 화면이 나타납니다.
4. **새 비밀번호**를 입력하고 **비밀번호 확인**란에 동일하게 입력합니다. (4자리 이상)
5. **개인정보 수집 및 이용 동의**:
   - **[필수]** 개인정보 수집 및 이용 동의: 성명, 이메일, 부서, 직책 수집에 동의합니다. 이 항목에 동의하지 않으면 서비스를 이용할 수 없습니다.
   - **[선택]** 개인정보 수집 및 이용 동의: 전화번호 수집에 동의합니다. 동의하지 않아도 서비스 이용에는 지장이 없습니다.
   - "전문 보기"를 클릭하면 약관 전문을 확인할 수 있습니다.
   - "전체 동의" 체크박스를 클릭하면 필수+선택 항목을 한 번에 동의할 수 있습니다.
6. **"시작하기"** 버튼을 클릭하면 비밀번호가 설정되고 자동으로 로그인됩니다.

### 이후 로그인
1. 이메일 주소를 입력하고 **"계속"**을 클릭합니다.
2. 설정한 비밀번호를 입력하고 **"로그인"**을 클릭합니다.

---

## 2. 대시보드

로그인 후 첫 화면입니다. **본인에게 배정된 장비만** 표시됩니다.

### 현황 카드
| 카드 | 설명 |
|------|------|
| 총 장비 | 본인에게 배정된 장비 수 |
| 준수율 | 배정 장비의 점검 기한 준수 비율 (%) |
| 점검 예정 | 7일 이내 점검 필요 장비 수 |
| 점검 지연 | 기한 초과 장비 수 |

### 차트 및 최근 활동
- 배정된 장비의 상태 분포와 최근 점검 기록이 표시됩니다.

---

## 3. 장비 관리

좌측 메뉴에서 **"장비 관리"**를 클릭합니다. 본인에게 배정된 장비만 표시됩니다.

### 3.1 장비 조회
- **검색**: 장비명 또는 추가정보로 검색
- **상태 필터**: 전체 / 정상 / 점검예정 / 점검지연 / 일시중지

### 3.2 점검 수행
1. 점검할 장비의 **"점검"** 버튼을 클릭합니다.
2. 오늘 날짜로 점검이 기록되고, 다음 점검 예정일이 자동으로 설정됩니다.
3. 다음 점검일이 주말에 해당하면 자동으로 다음 월요일로 조정됩니다.

> 점검 버튼을 클릭하면 즉시 처리됩니다. 점검 기한을 지켜주세요.

### 3.3 장비 정보 수정
- 장비의 **수정 아이콘**을 클릭하여 추가정보(시리얼번호), 메모 등을 수정할 수 있습니다.

### 3.4 장비 추가
- **"장비 추가"** 버튼을 클릭하여 새 장비를 등록할 수 있습니다.
- 장비명, 구분, 점검 주기 등을 입력합니다.

---

## 4. 접근 불가 메뉴 안내

담당자 역할에서는 다음 메뉴에 접근할 수 없습니다:

| 메뉴 | 설명 |
|------|------|
| 팀 관리 | 구분, 관리자, 사용자 관리는 마스터 또는 구분 관리자만 가능합니다 |
| 설정 | 시스템 설정은 마스터만 변경할 수 있습니다 |

> 해당 기능이 필요한 경우 구분 관리자 또는 마스터에게 문의하세요.

## 5. 이메일 알림

### 자동 알림 수신
시스템이 매일 오전 9시에 자동으로 점검 상태를 확인하여 알림 이메일을 보냅니다.

### 수신하는 알림 종류
| 알림 종류 | 시점 | 내용 |
|-----------|------|------|
| 점검 예정 알림 | 점검 예정일 7일 전부터 | 곧 점검이 필요한 장비 안내 |
| 점검 지연 알림 | 점검 예정일 경과 후 | 기한 초과 장비에 대한 긴급 안내 |

### 알림 이메일 예시
이메일에는 다음 정보가 포함됩니다:
- 장비명
- 구분
- 점검 예정일
- 점검 상태 (예정 / 지연)

> 알림 이메일은 등록된 이메일 주소로 발송됩니다. 이메일 주소 변경이 필요한 경우 마스터에게 문의하세요.

---

## 6. 자주 묻는 질문

**Q: 배정된 장비가 보이지 않습니다.**
A: 담당자는 본인에게 배정된 장비만 볼 수 있습니다. 장비가 배정되지 않았거나 다른 담당자에게 배정된 경우 구분 관리자에게 문의하세요.

**Q: 점검 버튼을 눌렀는데 다음 예정일이 이상합니다.**
A: 다음 점검 예정일은 점검 주기(일 단위)에 따라 자동 계산됩니다. 주말에 해당하면 월요일로 조정됩니다. 주기 변경이 필요하면 구분 관리자에게 문의하세요.

**Q: 비밀번호를 잊어버렸습니다.**
A: 마스터에게 비밀번호 초기화를 요청하세요. 초기화 후 다음 로그인 시 새 비밀번호를 설정할 수 있습니다.

**Q: 이메일 알림이 오지 않습니다.**
A: 등록된 이메일 주소가 정확한지 확인하세요. 스팸 메일함도 확인해 주세요. 문제가 계속되면 마스터에게 문의하세요.
