# AI 업무 알림 서비스 — 마스터(Admin) 매뉴얼

## 1. 로그인

1. 로그인 페이지에서 등록된 이메일 주소를 입력합니다.
2. "계속" 버튼을 클릭합니다.
3. 비밀번호를 입력하고 "로그인" 버튼을 클릭합니다.

> 마스터 계정의 비밀번호는 시스템 초기 설정 시 지정됩니다.

---

## 2. 대시보드

로그인 후 가장 먼저 표시되는 화면입니다. 시스템 전체 현황을 한눈에 파악할 수 있습니다.

### 상단 현황 카드
| 카드 | 설명 |
|------|------|
| 총 장비 | 시스템에 등록된 전체 장비 수 |
| 준수율 | 점검 기한을 지킨 장비의 비율 (%) |
| 점검 예정 | 향후 7일 이내에 점검이 필요한 장비 수 |
| 점검 지연 | 기한이 지났으나 점검되지 않은 장비 수 |

### 차트
- **원형 차트**: 장비 상태별 분포 (정상 / 점검예정 / 점검지연 / 일시중지)
- **막대 차트**: 구분(카테고리)별 장비 분포

### 최근 활동
- 최근 수행된 점검 기록과 시스템 변경 사항이 시간순으로 표시됩니다.

---

## 3. 장비 관리

좌측 메뉴에서 **"장비 관리"**를 클릭하면 전체 장비 목록이 표시됩니다.

### 3.1 장비 조회 및 필터링
- **검색**: 장비명 또는 추가정보(시리얼번호)로 검색
- **상태 필터**: 전체 / 정상 / 점검예정 / 점검지연 / 일시중지
- **구분 필터**: 특정 구분(카테고리)의 장비만 표시

### 3.2 장비 추가
1. 우측 상단 **"장비 추가"** 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
   - 장비명 (필수)
   - 구분 선택 (필수)
   - 추가정보 (시리얼번호 등)
   - 추가정보2 (메모)
   - 점검 주기 (일 단위, 구분의 기본 주기가 자동 설정됨)
   - 담당자 선택
   - 사용팀 선택
3. **"추가"** 버튼을 클릭하면 장비가 등록됩니다.

### 3.3 장비 수정
- 장비 목록에서 해당 장비의 **수정 아이콘**을 클릭합니다.
- 정보를 수정한 후 **"저장"**을 클릭합니다.

### 3.4 장비 삭제
- 장비 목록에서 해당 장비의 **삭제 아이콘**을 클릭합니다.
- 확인 팝업에서 **"삭제"**를 클릭합니다.

> 삭제된 장비는 복구할 수 없습니다. 일시적으로 관리에서 제외하려면 "일시중지"를 사용하세요.

### 3.5 점검 수행
- 장비의 **"점검"** 버튼을 클릭하면 오늘 날짜로 점검이 기록됩니다.
- 점검 주기에 따라 다음 점검 예정일이 자동 계산됩니다.
- 다음 점검일이 주말에 해당하면 자동으로 다음 월요일로 조정됩니다.

### 3.6 일괄 점검
1. 장비 목록에서 점검할 장비들의 **체크박스**를 선택합니다.
2. 상단에 나타나는 **"일괄 점검"** 버튼을 클릭합니다.
3. 선택한 모든 장비가 한 번에 점검 처리됩니다.

### 3.7 장비 일시중지 / 재개
- **일시중지**: 장비의 메뉴에서 "일시중지"를 선택하고 사유를 입력합니다. 일시중지된 장비는 점검 알림 대상에서 제외됩니다.
- **재개**: 일시중지된 장비의 "재개" 버튼을 클릭하면 다시 정상 관리 상태로 돌아갑니다.

## 4. 조직 관리 (팀 페이지)

좌측 메뉴에서 **"팀"**을 클릭합니다. 마스터는 4개의 탭에 모두 접근할 수 있습니다.

### 4.1 구분 탭
장비의 분류 체계를 관리합니다.

- **구분 추가**: "구분 추가" 버튼 → 구분명, 기본 점검 주기, 담당 관리자를 설정합니다.
- **구분 수정**: 구분 항목을 클릭하여 이름, 주기, 관리자를 변경합니다.
- **구분 삭제**: 해당 구분에 장비가 없는 경우에만 삭제할 수 있습니다.

> 구분의 관리자를 변경하면, 해당 구분에 속한 모든 장비의 관리자(managerId)가 자동으로 업데이트됩니다.

### 4.2 구분 관리자 탭
각 구분을 담당할 관리자를 지정합니다.

- 사용자를 **"관리자"** 역할로 승격하거나 해제할 수 있습니다.
- 관리자에게 담당 구분을 배정합니다.

### 4.3 사용자 관리 탭
시스템에 등록된 모든 사용자를 관리합니다.

- **사용자 추가**: 이름, 이메일, 부서, 소속팀, 직책을 입력합니다.
- **사용자 수정**: 목록에서 수정 아이콘을 클릭하여 정보를 변경합니다.
- **비밀번호 초기화**: 사용자의 비밀번호를 초기화하면, 해당 사용자는 다음 로그인 시 새 비밀번호를 설정하게 됩니다.
- **엑셀 업로드 (조직도 동기화)**: 엑셀 파일로 사용자 정보를 일괄 업데이트합니다.
  - 이 기능은 이름, 이메일, 부서, 직책 등 **사용자 기본 정보만** 동기화합니다.
  - 장비 배정 정보는 변경되지 않습니다.

### 4.4 마스터 탭
마스터(Admin) 계정을 관리합니다.

## 5. 설정

좌측 메뉴에서 **"설정"**을 클릭합니다. (마스터만 접근 가능)

### 알림 규칙
| 항목 | 설명 |
|------|------|
| 이메일 알림 | 점검 예정/지연 시 이메일 알림 발송 여부 |
| Slack 연동 | Slack 채널 알림 활성화 여부 |
| 지연 알림 | 점검 기한 초과 시 추가 알림 발송 여부 |

### 기본 설정
- **기본 점검 주기**: 새 장비 등록 시 적용되는 기본 주기 (일)
- **예정 알림 임계값**: 점검 예정일 며칠 전부터 "점검예정" 상태로 표시할지 설정 (기본: 7일)

---

## 6. 활동 로그

좌측 메뉴에서 **"로그"**를 클릭합니다.

- 시스템 전체의 점검 기록, 장비 변경, 사용자 활동이 시간순으로 표시됩니다.
- 각 로그에는 수행자, 대상 장비, 수행 일시, 메모가 포함됩니다.

---

## 7. 엑셀 활용 가이드

### 7.1 내보내기 (Export)
각 관리 화면에서 **"엑셀 내보내기"** 버튼을 클릭하면 현재 데이터가 엑셀 파일로 다운로드됩니다.

내보내기 가능 항목:
- 팀 목록
- 구분 목록
- 사용자 목록
- 장비 목록
- 점검 이력

### 7.2 템플릿 다운로드
**"템플릿 다운로드"** 버튼을 클릭하면 가져오기용 빈 양식 파일을 받을 수 있습니다.
- 템플릿의 컬럼 구조에 맞춰 데이터를 입력한 후 가져오기 하세요.

### 7.3 가져오기 (Import)
1. **"엑셀 가져오기"** 버튼을 클릭합니다.
2. 준비한 엑셀 파일을 선택합니다.
3. 미리보기 화면에서 데이터를 확인합니다.
4. **"가져오기"** 버튼을 클릭하면 데이터가 반영됩니다.

> 가져오기 결과로 추가/수정/삭제된 건수가 표시됩니다.

## 8. 이메일 알림

### 8.1 자동 알림
- 매일 오전 9시(KST)에 시스템이 자동으로 점검 상태를 확인합니다.
- **점검 예정 알림**: 7일 이내에 점검이 필요한 장비에 대해 담당자에게 이메일을 발송합니다.
- **점검 지연 알림**: 기한이 지난 장비에 대해 담당자 및 관리자에게 이메일을 발송합니다.

### 알림 수신 대상
| 수신자 | 설명 |
|--------|------|
| 담당자 (Staff) | 해당 장비에 배정된 담당자 |
| 팀장 | 담당자가 소속된 팀의 팀장 |
| 팀 대표 이메일 | 팀에 설정된 대표 연락 이메일 |
| 구분 관리자 | 해당 구분을 담당하는 관리자 |

### 8.2 수동 알림 발송
- 장비 관리 화면에서 특정 장비의 **"알림 발송"** 버튼을 클릭하면 해당 장비의 점검 알림을 즉시 발송할 수 있습니다.

### 8.3 테스트 이메일
- 설정 화면에서 **"테스트 이메일 발송"** 버튼을 클릭하면 Gmail API 연동 상태를 확인할 수 있습니다.
