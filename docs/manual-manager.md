# AI 업무 알림 서비스 — 구분 관리자(Manager) 매뉴얼

## 1. 로그인

### 최초 로그인 (비밀번호 미설정 시)
1. 로그인 페이지에서 등록된 이메일 주소를 입력합니다.
2. **"계속"** 버튼을 클릭합니다.
3. 최초 로그인 시 비밀번호 설정 화면이 나타납니다.
4. **새 비밀번호**를 입력하고 **비밀번호 확인**란에 동일하게 입력합니다. (4자리 이상)
5. **개인정보 수집 및 이용 동의**:
   - **[필수]** 개인정보 수집 및 이용 동의: 성명, 이메일, 부서, 직책 수집에 동의합니다. 이 항목에 동의하지 않으면 서비스를 이용할 수 없습니다.
   - **[선택]** 개인정보 수집 및 이용 동의: 전화번호 수집에 동의합니다. 동의하지 않아도 서비스 이용에는 지장이 없습니다.
   - "전문 보기"를 클릭하면 약관 전문을 확인할 수 있습니다.
   - "전체 동의" 체크박스를 클릭하면 필수+선택 항목을 한 번에 동의할 수 있습니다.
6. **"시작하기"** 버튼을 클릭하면 비밀번호가 설정되고 자동으로 로그인됩니다.

### 이후 로그인
1. 이메일 주소를 입력하고 **"계속"**을 클릭합니다.
2. 설정한 비밀번호를 입력하고 **"로그인"**을 클릭합니다.

---

## 2. 대시보드

로그인 후 첫 화면입니다. **본인이 담당하는 구분의 장비만** 표시됩니다.

### 현황 카드
| 카드 | 설명 |
|------|------|
| 총 장비 | 담당 구분에 속한 장비 수 |
| 준수율 | 담당 장비의 점검 기한 준수 비율 (%) |
| 점검 예정 | 7일 이내 점검 필요 장비 수 |
| 점검 지연 | 기한 초과 장비 수 |

### 차트 및 최근 활동
- 담당 구분 기준으로 필터링된 상태 분포 차트와 최근 점검 기록이 표시됩니다.

---

## 3. 장비 관리

좌측 메뉴에서 **"장비 관리"**를 클릭합니다. 본인이 관리하는 구분의 장비만 표시됩니다.

### 3.1 장비 조회
- **검색**: 장비명 또는 추가정보로 검색
- **상태 필터**: 전체 / 정상 / 점검예정 / 점검지연 / 일시중지
- **구분 필터**: 담당 구분 중 특정 구분만 표시

### 3.2 장비 추가
1. **"장비 추가"** 버튼을 클릭합니다.
2. 장비명, 구분, 추가정보, 점검 주기, 담당자 등을 입력합니다.
3. **"추가"**를 클릭합니다.

### 3.3 장비 수정
- 장비의 **수정 아이콘**을 클릭하여 정보를 변경합니다.

### 3.4 점검 수행
- 장비의 **"점검"** 버튼을 클릭하면 오늘 날짜로 점검이 기록됩니다.
- 다음 점검 예정일이 자동으로 계산됩니다. (주말인 경우 월요일로 조정)

### 3.5 일괄 점검
1. 여러 장비의 **체크박스**를 선택합니다.
2. **"일괄 점검"** 버튼을 클릭합니다.
3. 선택한 장비가 모두 점검 처리됩니다.

### 3.6 엑셀 내보내기 / 가져오기
- **내보내기**: 담당 구분의 장비 목록을 엑셀로 다운로드합니다.
- **가져오기**: 엑셀 파일로 장비 정보를 일괄 등록하거나 업데이트합니다.

---

## 4. 팀 관리

좌측 메뉴에서 **"팀"**을 클릭합니다.

> 구분 관리자는 **구분 탭**과 **배정 담당자 탭**에만 접근할 수 있습니다.

### 4.1 구분 탭
- 본인에게 배정된 구분의 정보를 확인하고 수정할 수 있습니다.
- 구분명, 기본 점검 주기를 변경할 수 있습니다.

> 다른 관리자가 담당하는 구분은 수정할 수 없습니다.

### 4.2 배정 담당자 탭
본인이 관리하는 장비의 담당자를 관리합니다.

- **담당자 배정**: 장비별로 담당자를 지정하거나 변경합니다.
- **엑셀 업로드 (장비별 담당자 동기화)**:
  1. **"템플릿 다운로드"**를 클릭하여 양식을 받습니다.
  2. 양식에 맞게 데이터를 작성합니다:
     - 컬럼: 구분, 대상(장비명), 담당자, 직책, 부서, 소속팀, 이메일, 전화번호
  3. **"엑셀 가져오기"**를 클릭하여 파일을 업로드합니다.
  4. 미리보기에서 데이터를 확인한 후 **"가져오기"**를 클릭합니다.

> 엑셀 가져오기 시 구분+대상(장비명) 조합으로 장비를 찾아서 담당자와 사용팀을 자동으로 변경합니다.

---

## 5. 활동 로그

좌측 메뉴에서 **"로그"**를 클릭합니다.

- 본인이 담당하는 구분의 장비에 대한 점검 기록과 변경 이력이 표시됩니다.

---

## 6. 이메일 알림

### 자동 알림 수신
- 매일 오전 9시에 시스템이 자동으로 점검 상태를 확인합니다.
- 담당 구분의 장비 중 점검 예정 또는 지연 상태인 장비가 있으면 이메일 알림을 받습니다.

### 알림 내용
- **점검 예정 알림**: 7일 이내에 점검이 필요한 장비 안내
- **점검 지연 알림**: 기한이 지난 장비에 대한 긴급 안내

---

## 7. 자주 묻는 질문

**Q: 다른 관리자의 구분에 속한 장비가 보이지 않습니다.**
A: 구분 관리자는 본인에게 배정된 구분의 장비만 볼 수 있습니다. 다른 구분의 장비를 확인해야 하는 경우 마스터에게 문의하세요.

**Q: 담당자를 변경했는데 반영이 안 됩니다.**
A: 페이지를 새로고침한 후 다시 확인해 주세요. 문제가 계속되면 마스터에게 문의하세요.

**Q: 비밀번호를 잊어버렸습니다.**
A: 마스터에게 비밀번호 초기화를 요청하세요. 초기화 후 다음 로그인 시 새 비밀번호를 설정할 수 있습니다.
