스케줄 관리시스템의 목적과 구성

📅

점검 스케줄 관리

대상별 점검 주기를 설정하고 다음 점검 예정일을 자동 계산하여 체계적으로 관리합니다.

📧

자동 이메일 알림

매일 오전 9시(KST) 점검 예정 및 지연 현황을 담당자에게 자동으로 이메일 발송합니다.

👥

역할 기반 관리

3가지 역할로 권한을 분리하여 안전하고 효율적인 운영 체계를 제공합니다.

👑 마스터 전체 시스템 관리 및 설정

👤 담당자 배정된 대상 점검 실행

로그인 완료
  • 마스터가 사전 등록한 이메일로만 로그인 가능
  • 최초 로그인 시 비밀번호 설정 화면이 자동 표시됨
  • 비밀번호는 4자리 이상으로 설정
  • 비밀번호 설정 완료 후 자동으로 로그인 처리

💡 마스터 팁: 사용자 등록 시 이메일 주소를 정확히 입력해야 해당 사용자가 로그인할 수 있습니다. 비밀번호 분실 시 마스터가 초기화할 수 있습니다.

대시보드

전체 대상 현황을 한눈에 확인

주요 현황 카드

  • 전체 대상 수 — 시스템에 등록된 전체 대상 합계
  • 점검 준수율 — 주기 내 점검 완료 비율 (%)
  • 점검 임박 — 7일 이내 점검 예정 대상 수
  • 점검 지연 — 예정일 초과 대상 수 (즉시 조치 필요)

상태별 분류

🟢 정상 — 점검 주기 내 완료
🟡 임박 — 7일 이내 점검 예정
🔴 지연 — 점검 예정일 초과

📈 준수율 차트와 구분별 분포 그래프로 현황을 시각적으로 파악

🏷️
대상 및 사용자 관리

대상 탭

대상(카테고리) 등록 및 관리자 배정

  • 대상 등록 — 대상명을 입력하여 새 대상(카테고리) 생성
  • 대상 관리자 배정 — 각 대상에 담당할 대상 관리자를 지정
  • 수정 / 삭제 — 기존 대상의 이름 변경 또는 삭제
  • 엑셀 다운로드 — 현재 대상 목록을 엑셀 파일로 내보내기
  • 엑셀 업로드 — 엑셀 파일로 대상을 일괄 등록

💡 대상이란? 점검이 필요한 항목들을 분류하는 카테고리입니다. 예: 계량기, 검사장비, 검사차량 등. 스케줄 관리에서 개별 항목 등록 시 이 대상을 선택합니다.

🔧
대상 및 사용자 관리

대상 관리자 탭

대상 관리자 등록 및 관리

  • 대상 관리자 배정 — 일반 사용자(담당자)를 대상 관리자 역할로 승격
  • 관리자 목록 확인 — 이름, 소속팀, 이메일, 전화번호, 로그인 상태 조회
  • 정보 수정 — 대상 관리자의 기본 정보 변경
  • 비밀번호 초기화 — 로그인 비밀번호를 초기 상태로 리셋
  • 관리자 해제 — 대상 관리자 역할을 해제하여 담당자로 변경

💡 대상 관리자 역할: 자신이 배정된 대상의 담당자와 스케줄을 관리합니다. 마스터가 관리 범위를 위임하는 중간 관리자입니다.

👥
대상 및 사용자 관리

전체 사용자 탭

담당자(직원) 계정 등록 및 관리

사용자 등록 항목

  • 이름 — 사용자 실명
  • 직책 — 팀장, 팀원 등
  • 소속팀 — 팀 선택
  • 이메일 — 로그인 및 알림 수신용
  • 전화번호 — 연락처 (선택)

관리 기능

  • 사용자 수정 — 정보 변경 및 업데이트
  • 사용자 삭제 — 계정 제거
  • 비밀번호 초기화 — 다음 로그인 시 재설정
  • 엑셀 다운로드 — 사용자 목록 내보내기
  • 엑셀 업로드 — 사용자 일괄 등록
👑
대상 및 사용자 관리

마스터 탭

추가 마스터 계정 생성 및 관리

  • 마스터 계정 추가 생성 — 시스템 관리를 분담할 추가 마스터 등록
  • 마스터 권한 — 전체 시스템의 모든 기능에 대한 완전한 접근 권한
  • 마스터 목록에서 기존 마스터 계정 확인 가능
🛡️

마스터 권한 범위

대상/사용자 관리, 스케줄 관리, 대상 관리자 배정, 시스템 설정 등 모든 기능 사용 가능

⚠️

주의사항

마스터 계정은 최소한의 인원에게만 부여하세요. 마스터는 사용자 삭제, 데이터 변경 등 민감한 작업이 가능합니다.

📋

스케줄 관리

점검 대상 항목 등록 및 점검 실행

항목 등록 정보

  • 대상명 — 점검 대상의 이름
  • 일련번호 — 고유 식별 번호
  • 대상(카테고리) — 분류 선택
  • 담당팀 / 사용팀 — 관련 팀 지정
  • 담당자 — 점검 실행 담당자
  • 점검주기 — 점검 간격 (개월)
  • 최근점검일 — 마지막 점검 날짜

관리 기능

  • 점검 실행 — 점검일 기록, 다음 예정일 자동 계산
  • 항목 수정 — 등록 정보 변경
  • 항목 삭제 — 대상 항목 제거
  • 엑셀 다운로드 — 전체 목록 내보내기
  • 엑셀 업로드 — 항목 일괄 등록
  • 상태/구분별 필터 — 원하는 조건으로 조회
