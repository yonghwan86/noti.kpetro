📋

시스템 개요

시스템의 목적과 담당자의 역할

스케줄 관리시스템은 다양한 대상(장비, 시설 등)의 정기 점검 스케줄을 체계적으로 관리하는 시스템입니다.
🎯

시스템 목적

점검 대상의 주기적 점검을 빠짐없이 수행하여 안전성과 신뢰성을 확보합니다.

👤

담당자 역할

배정받은 대상의 점검을 직접 수행하고 점검 결과를 시스템에 기록합니다.

📊

핵심 업무

점검 일정 확인, 점검 실행, 결과 기록 — 이 세 가지가 담당자의 핵심 업무입니다.

🔐

로그인

시스템 접속 방법

📧 로그인 방법

  • 마스터 또는 대상 관리자가 사전 등록한 이메일로 로그인
  • 이메일 주소를 입력하면 등록 여부를 자동으로 확인
  • 등록되지 않은 이메일은 로그인 불가

🔑 최초 로그인

  • 처음 로그인 시 비밀번호 설정 화면이 표시됩니다
  • 4자리 이상의 비밀번호를 설정하세요
  • 설정 완료 후 자동으로 로그인됩니다
💡 이메일이 등록되지 않은 경우 대상 관리자에게 등록을 요청하세요.
📊

대시보드

나에게 배정된 대상의 현황 한눈에 보기

  • 로그인 후 담당자 대시보드가 표시됩니다
  • 나에게 배정된 대상의 전체 수, 점검 준수율, 임박/지연 현황을 확인
  • 각 카드를 클릭하면 해당 상태의 대상 목록으로 이동

상태 구분

✅ 정상 — 점검 주기 내 완료
⏰ 점검 임박 — 7일 이내 예정
⚠️ 점검 지연 — 예정일 초과
📌 대시보드에는 나에게 배정된 대상만 표시됩니다. 다른 담당자의 대상은 보이지 않습니다.
📅

스케줄 관리

배정된 대상의 점검 실행

📋 목록 확인

  • 나에게 배정된 대상만 목록에 표시
  • 대상명, 구분, 최근 점검일, 다음 예정일 확인
  • 상태 필터로 정상/임박/지연 구분 조회
  • 검색 기능으로 특정 대상 빠르게 찾기

✅ 점검 실행

  • 점검 버튼을 클릭하여 점검 기록
  • 점검 실시일을 선택 후 확인
  • 점검일 기록 시 다음 점검일이 자동 계산
  • 점검 주기(개월)에 따라 자동 산출
🚫 담당자는 대상 정보의 수정 및 삭제가 불가합니다. 수정이 필요하면 대상 관리자에게 요청하세요.
📧

이메일 알림

자동 점검 알림 시스템

⏰

매일 자동 알림

매일 오전 9시(KST)에 점검 현황을 자동으로 확인하여 이메일을 발송합니다.

📅

점검 예정 알림

점검일 7일 전부터 예정 알림을 수신합니다. 미리 점검 준비를 하세요.

🚨

지연 경고 알림

점검 예정일이 지나면 경고 알림이 발송됩니다. 즉시 점검을 수행하세요.

💡 알림은 담당자 본인과 대상 관리자에게도 함께 발송됩니다. 점검을 미루지 마세요!
